DOGE, BIT, and ADA Tracker

Console based CrytoTracker for Dogecoin, Bitcoin, and Cardano.

This program will simply ask for your owned amount of DOGE, BIT, and ADA then stores your coin within a JSON file for persistant storage. The program will scrape Coin Market Cap for the current price of the coins and shows the user their total investment. Refreshes every five minutes.

Language Used:
Python 3.19

Modules:
BeautifulSoup, OS, and Time

Limitations:
Program is only able to track the 3 coins

Exporting:
None - Console only
